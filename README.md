# DETERMINANT CALCULATOR

A web-based linear algebra tool designed to calculate the determinant of square matrices ranging from 1x1 to 10x10 matrices.

This project is built as a course requirement for MATH 206 - Linear Algebra

* * * * *

### FEATURES

-   User-defined input: Have the user input their desired matrix.

-   Wide range: Handles a wide range of square matrices (1x1 to 10x10).

-   Efficient Calculation: The program instantly calculates the determinant of a given square matrix.

-   User-friendly UI: The site offers a simple-to-use UI with a help page to guide users on how to use the site.

### BUILT WITH

-   HTML: Used for the structure and layout of the site.

-   CSS: Used for the styling and responsive design of the site.

-   JavaScript: Used for the logic of the determinant calculator.

* * * * *

### HOW TO USE

1.  Enter the size you want the matrix to be. 

    *Note: the system can only handle up to 10 by 10 matrices.*

2.  After entering the size you wanted, click on the Generate Matrix button to have it generated.

3.  Fill in the values you want for your matrix in the display.

4.  Click the Calculate Determinant button after filling out the necessary values needed.

5.  Scroll down the site, and you'll be greeted by the determinant of the matrix you've created. You can click the reset button if you want to calculate again.
